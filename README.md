# Online Judge 前端

## 环境安装
```shell
npm install
```

## 代码运行
```shell
npm run dev
```

## 目录结构说明

```
src
├── api         与后端通信的接口✨
├── assets      资源文件夹
├── components  功能组件✨
├── layouts     视图的布局
├── pages       用户能直接看到的页面✨
├── plugins     插件
├── router      路由
├── stores      浏览器本地存储✨
└── styles      样式
```
